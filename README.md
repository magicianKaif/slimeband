## Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/SlimeBand.git
   ```

2. Navigate to the project directory:
   ```bash
   cd SlimeBand
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm start
   ```

5. Open your browser and visit:
   ```
   http://localhost:3000
   ```
